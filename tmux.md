# Tmux

## Intro

Tmux utilise une pré-commande : [CTRL] + [b]

## Switch vers d'autres panels

[CTRL] + [b] puis [o]

## chercher une fenêtre/panel puis switcher :

[CTRL] + [b] puis [f]

## détacher

[CTRL] + [b] puis [d]

## supprimer

[CTRL] + [b] puis [&]
